# Video Demo Prompts

Use these for same-day recording.

## Demo 1: Read and summarize a codebase

Prompt:

```text
先浏览当前项目结构，找到入口文件，读关键文件后用中文总结这个项目是做什么的，再给我 3 个最值得拍成视频的技术点。
```

Why it works:

- Immediately shows directory browsing
- Triggers multiple local tools
- Produces content-friendly output

## Demo 2: Make a visible code change

Prompt:

```text
在当前项目里找一个 README 或示例脚本，增加一个更清晰的“快速开始”区块，然后解释你改了什么。
```

Why it works:

- Lets viewers see a real file edit
- Easy to verify on screen
- Low risk for live recording

## Demo 3: Turn the shell into a creator assistant

Prompt:

```text
根据这个项目，帮我写一段适合短视频口播的 90 秒脚本，语气要像技术热点拆解，结尾给一个强钩子。
```

Why it works:

- Converts the tool from dev toy to creator workflow
- Matches your use case directly

## Demo 4: Hot topic framing

Prompt:

```text
把这个项目包装成一个“今天能讲清楚”的热点选题，给我 5 个标题、1 个封面文案、1 个视频结构大纲。
```

Why it works:

- Good for immediate publishing
- Gives title and packaging assets fast

## Suggested on-camera framing

- Do not claim this is the official Anthropic client running unmodified.
- Say it is a compatible shell rebuilt from a recovered source slice and wired to external models.
- Emphasize that the interesting part is the architecture and workflow, not brand cosplay.
