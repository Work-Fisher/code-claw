# WorkFisher Brand Styling

## Overview

To access WorkFisher's official brand identity and style resources, use this skill.

**Keywords**: branding, corporate identity, visual identity, styling, brand colors, typography, visual formatting, visual design

## Brand Guidelines

### Colors (WorkFisher Dark Cinematic Theme)

**Main Colors:**

- Dark Background: `#0c0a10` - Primary background
- Panel Surface: `#161222` / `#1a1528` - Card/panel backgrounds
- Light Text: `#f0eef4` - Primary text on dark
- Mid Gray: `#b0aea5` - Secondary elements
- Muted Text: `#8a86a0` - Subdued labels

**Accent Colors:**

- Magenta (Primary): `#f03ca4` — `var(--accent)`, CTAs, highlights, active borders
- Magenta Bright: `#ff61c4` — `var(--accent-strong)`, hover states, active text
- Magenta Dim: `rgba(240, 60, 164, 0.15)` — subtle accent backgrounds

### Typography

- **Headings**: System sans-serif stack (bold weight)
- **Body Text**: System sans-serif stack (regular weight)
- **Monospace**: For code/technical displays

### Design Principles

- Dark cinematic aesthetic — NOT blue-purple sci-fi
- Black-gray glass panels with subtle transparency
- Magenta/pink accent (`#f03ca4`) throughout — NOT orange, NOT purple
- Generous spacing and breathing room
- NO neon effects, NO purple gradients on white
- Subtle radial gradients for depth
