# Compatible Agent Shell Alpha

This package is an early alpha for technical testers.

What it is:

- a local coding shell inspired by Claude Code-style workflows
- a local Anthropic-compatible gateway that can sit in front of OpenAI-compatible model APIs
- a practical prototype for code reading, searching, editing, and controlled command execution

What it is not:

- not the official Anthropic Claude Code client
- not a byte-perfect restoration of any leaked or recovered project
- not a polished consumer product

## Current features

- browse directories
- read files
- write files
- search text
- run shell commands with approval

## Requirements

- Windows
- Node.js 22 or newer
- your own OpenAI-compatible API endpoint

## Recommended wording when sharing

Use language like:

- "Claude-inspired compatible shell"
- "agent shell alpha"
- "compatible coding shell backed by your own model endpoint"

Avoid language like:

- "official Claude Code"
- "Claude Code fully restored"
- "same as Anthropic's internal product"
